## 👋 Introdução

Bem-vindo à **Galeria de Fotos & Portfólio**, uma plataforma moderna e profissional desenvolvida pela equipe da [Invillia](https://www.invillia.com) para fotógrafos, criadores de conteúdo e profissionais visuais que desejam exibir seu trabalho com elegância e praticidade.

Esta aplicação foi construída com as mais modernas tecnologias do ecossistema JavaScript — **Next.js 15**, **TypeScript** e **Tailwind CSS** — e serve também como vitrine das capacidades do **GitHub Copilot** no desenvolvimento de software orientado a componentes.

---

## 🎯 Missão

Nossa missão é **democratizar o acesso a ferramentas profissionais de portfólio**, oferecendo uma plataforma intuitiva, rápida e visualmente atraente que permita a qualquer fotógrafo ou criador apresentar seu trabalho ao mundo — sem complicações técnicas.

---

## 🌟 Visão

Ser referência em **ferramentas de portfólio digital** para criativos no Brasil e no mundo, promovendo a inovação contínua com o apoio da inteligência artificial e das melhores práticas de desenvolvimento de software.

## 📖 Como Testar Localmente

> **Instruções para o revisor:**

1. **Visualizar este arquivo Markdown:**
   - Abra o arquivo `about.md` diretamente no GitHub ou em qualquer editor com preview Markdown (ex.: VS Code com a extensão Markdown Preview).
   - No VS Code, pressione `Ctrl+Shift+V` (Windows/Linux) ou `Cmd+Shift+V` (macOS) para abrir o preview.

2. **Rodar a aplicação localmente:**
   ```bash
   npm install
   npm run dev
   ```
   Em seguida, acesse [http://localhost:3000](http://localhost:3000) no navegador.

3. **Sugestões de edição:**
   - Caso prefira uma página HTML em vez de Markdown, informe nos comentários do PR.
   - Se o repositório adotar um diretório específico de páginas (ex.: `/docs` ou `/pages`), o arquivo pode ser movido após revisão.
   - Sinta-se à vontade para propor ajustes no texto, nos emojis ou na estrutura das seções.
